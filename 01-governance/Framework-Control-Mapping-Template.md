# Framework Control Mapping Template

## Purpose
This template maps repository controls to external frameworks for alignment and traceability.

This is an alignment aid only. It does not provide certification, legal advice, or a claim of formal compliance.

## Frameworks Included
- Gartner AI TRiSM (conceptual operating model reference)
- NIST AI RMF (Govern, Map, Measure, Manage)
- ISO/IEC 42001 (AI management system requirements)
- EU AI Act (optional, where applicable)

## Mapping Table
| Local Control ID | Local Control Objective | TRiSM Concept | NIST AI RMF Function | ISO/IEC 42001 Clause | EU AI Act Article or Obligation | Evidence Artifact | Control Owner | Notes |
|---|---|---|---|---|---|---|---|---|

## Mapping Rules
- Keep one control per row.
- Map to external references only where the intent is materially similar.
- Use "N/A" where no equivalent reference exists.
- Link at least one evidence artifact for each local control.
