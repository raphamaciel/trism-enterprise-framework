# Deployment Gate Checklist Template

## Gate 1 - Design Assurance
- [ ] Threat model completed and reviewed
- [ ] Control matrix mapped for deployment scope
- [ ] Architecture decisions documented

## Gate 2 - Pre-Production Assurance
- [ ] Risk register updated with treatment owners
- [ ] Model validation report completed
- [ ] Security and privacy evidence attached

## Gate 3 - Go-Live Assurance
- [ ] High and critical findings resolved or formally accepted
- [ ] Incident response playbook validated
- [ ] Required approvers signed off

## Gate 4 - Post-Live Assurance
- [ ] Monitoring KPIs/KRIs configured
- [ ] Drift and anomaly alerts active
- [ ] First post-live review date scheduled
